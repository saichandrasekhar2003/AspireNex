Project 1

Student Attendance Management System Android Application

(Flutter, FireBase, SpringBoot, Dart)

Developed using Flutter, Spring Boot, and Firebase, SAMS is a comprehensive software designed to streamline the management of academic and administrative tasks in educational institutions. It offers a user-friendly interface for both teachers and students to manage attendance efficiently.

Key Features:

  • Time Efficiency: Unlike traditional offline systems, SAMS saves time and reduces manual data entry.
  • Data Security: Attendance records are securely stored in a database, ensuring easy access and protection against data loss.
  • Geolocation: Only students present in the classroom can record attendance using geolocation features.

Technologies Used:

  • Frontend: Flutter (Android)
  • Backend: Spring Boot
  • Database: Firebase

This project exemplifies my ability to develop integrated solutions using modern technologies, ensuring both efficiency and security in data management.
